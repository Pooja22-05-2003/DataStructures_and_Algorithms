Given 2 numbers A,B and C, you need to find the minimum possible value of X, which satisfy the expression
(A|X)&(B|X)=C

If there is no possible value of X which satisfy the expression, then return -1.

Constraints:
1<=A,B,C<=10^9

Example :
3 2 1

Output :
-1 (X value is not possible.)
